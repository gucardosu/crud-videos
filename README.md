# crud-videos

Projeto simples de API REST desenvolvido com Node.js para realizar operações de CRUD (Create, Read, Update, Delete) de vídeos, utilizando um banco de dados local para armazenamento das informações.

## Descrição

A aplicação permite gerenciar vídeos através de uma API. É possível cadastrar novos vídeos, listar vídeos existentes, atualizar informações e remover registros. O objetivo do projeto é praticar conceitos básicos de desenvolvimento back-end com Node.js e manipulação de banco de dados local.

## Tecnologias utilizadas

- Node.js  
- Express.js  
- JavaScript  
- Banco de dados local (SQLite ou arquivo JSON)

## Funcionalidades

- Criar um novo vídeo
- Listar todos os vídeos
- Buscar vídeo por ID
- Atualizar informações de um vídeo
- Deletar um vídeo
